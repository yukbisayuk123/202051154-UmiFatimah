Cara Upload File di 000WebHost

  • Login ke 000WebHost
  • Setelah masuk dasbor admin, cari dan klik menu FILE MANAGER
  • Klik tombl upload file now
  • Seetelah itu kamu akan dibawa ke laman baru yang berisi direktori website kamu. Disinilah kamu bisa mengupload file yang kamu butuhkan
  • Untuk masuk ke dalam sebuah folder, klik dua kali pada folder yang akan kamu akses. Bisanya kamu akan diminta memasukkan nama website dan password terlebih dahulu
  • Untuk mengupload file ke 000WebHost, cari menu UPLOAD FILES pada menu bar di atas dan pilih file yang akan kamu upload
  • Setelah itu tunggu sampai proses upload selesai
